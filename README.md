# Fastreset

Reset css for Fastcampus

## Table Contents

* [Project setup](#Project-setup)
* [설치](#설치)
* [사용하기](#사용하기-Vuejs)

## Project setup

```console
$ npm install
```

#### Compile scss to css

```console
$ npm run build
```

## 설치

```console
$ npm install --save @fastcampus/fastreset
```

## 사용하기

### 로드하기

```console
@import '~@fastcampus/fastreset'; load bundled css
or 
@import '~@fastcampus/fastreset/src/scss/reset'; load specific scss file
```
